Kerrax ASC Exporter для 3D Studio Max 5
---------------------------------------

Этот плагин к 3D Studio Max позволяет экспортировать ASC-файлы
моделей к Готике 1-2. Имеет те же функции, что и плагин ZenGin ASC Exporter.
Написан по причине неработоспособности ZenGin ASC Exporter для
версий 3D Studio Max старше 5.

Установка (в предположении, что 3D Studio Max установлен в папку C:\3dsmax):
1) выйти из 3D Studio Max, если он запущен
2) скопировать krxexp.dle в папку C:\3dsmax\stdplugs
3) удалить файл krxexp.cfg (и только его!) 
из папки C:\3dsmax\plugcfg, если он там есть.
4) все, можно запускать 3D Studio Max снова.
