
GothicZTEX 1.2
==============

What for...
¯¯¯¯¯¯¯¯¯¯¯
This tool is intended to convert the compressed
textures of the game Gothic ( vers. 1.08 ) back
to TGA files ( including directories ).

Preparation
¯¯¯¯¯¯¯¯¯¯¯
At first expand the following VDFS volumes to
your harddrive ( if not already done ):
[Gothic]\Data\Fonts.vdf
[Gothic]\Data\Textures.vdf
[Gothic]\Data\Textures_Patch.vdf

note: you may use GothicVDFS for this task.

Installation
¯¯¯¯¯¯¯¯¯¯¯¯
Copy the two files ( exe/ini ) into the textures
directory ( [Gothic]\_Work\Data\Textures ).
and simply press the "DO IT" button =)

Updates?
¯¯¯¯¯¯¯¯
Don't ask for =)
There is another project running which
will be able to do the same work...
IMHO GothicZTEX is ( and ever was ) an
temporary solution to help a little bit.
